Style Transfer using CycleGAN

Code | Demo
Deep Gandhi

Developed a CycleGAN-based image style transfer model to translate between Van Gogh-style paintings and real-world photographs. Trained on the VanGogh2Photo dataset (800+ images), the model successfully generated visually compelling and stylistically accurate transformations.

Eye Disease Classification with XAI Techniques

Code | Demo
Deep Gandhi

This project tackles eye disease classification using pretrained models like ResNet, ViT, and InceptionV3, enhanced with fine-tuning techniques such as gradual unfreezing and spatial transformers. To ensure model transparency, we integrated XAI methods including Grad-CAM, Grad-CAM++, Smooth Grad-CAM++, and CRP, enabling interpretable and clinically reliable predictions.
